DOM

O Document Object Model - DOM é uma API multiplataforma que permite o acesso através do JavaScript a todos os elementos HTML que fazem parte da nossa página.

A organização do documento sergue uma representação de uma árvore (Árvore binária), no qual visa estabelecer trajetos específicos para alcançar cada um dos elementos que constituem a página.
