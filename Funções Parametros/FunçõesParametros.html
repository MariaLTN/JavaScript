Flexibilidade dos Parametros

function soma (a,b){
return a+b;
}

Nesse caso, é esperado que passemos dois parametros para a função soma.

Se usarmos a função: soma(7,7) ele retornará o número 14. O previsto.

Mas caso passemos mais parametros: soma(7,7,2,3). Ele retornará apenas o 14 e descartará os outros dois números (2,3)

Mas caso passemos parametros a menos: soma(7). Ele interpretará: 7 + undefined = not a number (NaN)

E caso não passemos nenhum valor: soma() ele retorna um Not a Number(NaN)
